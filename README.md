# Car Roadshow Website

This project is a **Car Roadshow Event Website** that allows users to register for the roadshow and enables administrators to manage the registrations via an admin panel.

---

# Live Website

You can view the registration form here:

**Main Website:**  
https://car-roadshow.is-great.net/index.php

---

# Admin Panel

The admin panel allows administrators to manage customer registrations.

**Admin Login:**  
https://car-roadshow.is-great.net/car_admin.php

# Features

- User-friendly web interface for participants
- Car roadshow registration form
- Admin panel for managing registrations
- Dynamic updates through the admin dashboard
