# Car Roadshow Registration System

This project is a Car Roadshow Event Website that allows users to register for the roadshow and enables administrators to manage the registrations via an admin panel. It showcases CRUD operations, file uploads, and admin dashboard development.

## 🔗 Live Demo
- **User Registration:** [Visit Form](https://car-roadshow.is-great.net/index.php)
- **Admin Panel:** [Login Here](https://car-roadshow.is-great.net/car_admin.php)

## 🔑 Demo Admin Access

*Please do not modify the example data directly through the admin panel.*
- **Username:** Car
- **Password:** Carevent
## ✨ Features

### For Users:
- Simple registration form
- Driving license upload functionality
- Confirmation upon successful registration

### For Admins:
- Secure login system
- View all customer registrations
- View, refresh, delete, and filter entries

## 🛠️ Technologies Used
- **Backend:** PHP
- **Frontend:** HTML5, CSS3, JavaScript
- **Database:** MySQL
- **Additional:** File upload handling, session management

## 📁 Project Structure
| File / Folder | Description |
|---------------|-------------|
| index.php | Main website page (registration form) |
| car_admin.php | Admin login page |
